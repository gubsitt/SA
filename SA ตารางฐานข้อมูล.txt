1. Entities และ Attributes

User (ผู้ใช้)/

UserID (Primary Key)
Username
Password
Email
Role (ทั่วไป/สมาชิกครอบครัว)

Category (หมวดหมู่)/

CategoryID (Primary Key)
CategoryName
Description

Transaction (รายการ)/

TransactionID (Primary Key)
UserID (Foreign Key)
CategoryID (Foreign Key)
Amount
Date
Type (รายรับ/รายจ่าย)
Description

Budget (งบประมาณ)/

BudgetID (Primary Key)
UserID (Foreign Key)
CategoryID (Foreign Key)
Amount
StartDate
EndDate
Frequency (รายสัปดาห์/รายเดือน/รายปี)

SavingsGoal (เป้าหมายการออม)/

GoalID (Primary Key)
UserID (Foreign Key)
TargetAmount
StartDate
EndDate
ProgressAmount

Bill (บิลประจำงวด)/

BillID (Primary Key)
UserID (Foreign Key)
CategoryID (Foreign Key)
Amount
DueDate
Status (Paid/Unpaid)

Account (บัญชี)/

AccountID (Primary Key)
UserID (Foreign Key)
AccountName
Balance
AccountType (เช่น บัญชีออมทรัพย์, บัตรเครดิต, กระเป๋าเงิน)
CreatedDate


2.Relationships
User to Transaction: One-to-Many (1 User สามารถมีหลาย Transactions)
User to Budget: One-to-Many (1 User สามารถมีหลาย Budgets)
User to SavingsGoal: One-to-Many (1 User สามารถมีหลาย SavingsGoals)
User to Bill: One-to-Many (1 User สามารถมีหลาย Bills)
User to Account: One-to-Many (1 User สามารถมีหลาย Accounts)
Category to Transaction: One-to-Many (1 Category สามารถมีหลาย Transactions)
Category to Budget: One-to-Many (1 Category สามารถมีหลาย Budgets)
Category to Bill: One-to-Many (1 Category สามารถมีหลาย Bills)
Account to Transaction: One-to-Many (1 Account สามารถมีหลาย Transactions)
Account to SavingsGoal: One-to-Many (1 Account สามารถมีหลาย SavingsGoals)
Account to Bill: One-to-Many (1 Account สามารถมีหลาย Bills)
